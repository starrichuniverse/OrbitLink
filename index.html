OrbitLink 개인정보처리방침

최종 수정일: 2026년 5월 20일

1. 수집하는 정보

OrbitLink는 다음의 정보만을 처리합니다:

  • Google 계정 이메일: 로그인 인증 목적으로만 사용하며, 외부로 전송하지 않습니다.
  • 인증 토큰(JWT): 브라우저 로컬 스토리지에만 저장되며, 서드파티에 공유되지 않습니다.
  • 북마크 및 매핑 설정: 브라우저 로컬 스토리지에만 저장됩니다.
  • 팝업 진행 상태: 최근 선택한 단계와 입력값 일부를 브라우저 로컬 스토리지에 저장할 수 있습니다.
  • 탭/창 문맥 정보: 현재 탭 또는 마지막 활성 창의 업무 탭 식별에 한해 일시적으로 사용하며, 브라우징 기록 수집 목적으로 저장하지 않습니다.

2. 정보 이용 목적

  • Supabase를 통해 사내 보험 상품 데이터를 조회합니다.
  • Google OAuth를 통한 사내 구성원 인증에만 사용합니다.
  • 관리자가 공개 모드를 활성화한 경우 일반 사용자가 로그인 없이 팝업에 접근할 수 있도록 합니다.
  • 콘텐츠 스크립트는 사용자가 팝업에서 명시적으로 실행한 경우에만 현재 탭에 동적으로 주입하며, 값 조회와 입력 실행도 해당 시점에만 수행합니다.
  • host_permissions는 Supabase API 도메인 2개로 제한되며 localhost, 127.0.0.1, broad host 권한을 사용하지 않습니다.
  • web_accessible_resources의 <all_urls> 공개를 사용하지 않습니다.
  • 수집된 정보를 마케팅, 광고, 제3자 제공에 활용하지 않습니다.

3. 외부 서비스 연동

  • Supabase (supabase.co): 상품 데이터베이스 및 인증 처리. Supabase 개인정보처리방침
  • Google OAuth: 로그인 인증 전용. Google 개인정보처리방침

4. 공개 모드 안내

관리자가 설정에서 공개 모드를 활성화하면 일반 사용자는 로그인 없이도 확장 프로그램 본 화면에 접근할 수 있습니다. 이 상태에서는 관리자 설정 변경 기능을 사용할 수 없으며, 관리자만 별도 로그인 버튼을 통해 인증 후 설정을 변경할 수 있습니다.

5. 데이터 보관 및 삭제

로그아웃 시 로그인 세션 토큰은 즉시 삭제됩니다. 페이지 매핑, 공개 모드 설정, 팝업 진행 상태와 같은 로컬 설정은 사용자 편의를 위해 브라우저 로컬 저장소에 남을 수 있으며, Chrome 확장 프로그램 삭제 시 함께 제거됩니다.

6. 대상 사용자

본 확장 프로그램은 사내 인가된 구성원 전용이며, 일반 공개 배포를 목적으로 하지 않습니다.

7. 권한 및 실행 원칙

  • activeTab, scripting: 사용자가 현재 보고 있는 페이지에서 명시적으로 실행한 경우에만 입력 보조를 수행합니다.
  • tabs, windows: 현재 창 또는 마지막 활성 창 범위에서 대상 업무 탭을 식별하는 용도로만 사용합니다.
  • identity: 관리자 설정 변경을 위한 Google OAuth 인증에만 사용합니다.
  • storage: 세션/공개 모드/매핑/팝업 상태 저장에만 사용하며 외부 제3자에게 판매하거나 공유하지 않습니다.
