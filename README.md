# OrbitLink Privacy Policy Page

이 저장소는 OrbitLink Chrome 확장 프로그램의 개인정보처리방침 공개 페이지 전용입니다.

## Public URL

- https://starrichuniverse.github.io/OrbitLink/

## Files

- index.html: Chrome Web Store 제출용 개인정보처리방침 본문
- .nojekyll: GitHub Pages 루트 정적 배포 보조 파일

## Update flow

1. planet 저장소의 apps/extension/privacy_policy.html 변경사항을 확인합니다.
2. 이 저장소의 index.html에 동일 내용을 반영합니다.
3. main 브랜치에 push 합니다.
4. GitHub Pages 배포 후 공개 URL 응답을 확인합니다.

## GitHub Pages setup

1. Settings > Pages 로 이동합니다.
2. Build and deployment 에서 Deploy from a branch 를 선택합니다.
3. Branch 는 main, Folder 는 /(root) 로 설정합니다.
4. 저장 후 https://starrichuniverse.github.io/OrbitLink/ 접속을 확인합니다.
